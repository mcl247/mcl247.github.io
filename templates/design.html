PROGRAMMING AND DESIGNING USER-FRIENDLY WEBPAGES


Atlanta Noms

Worked in a team of 3 to create a food blog-style website for Atlanta based client using PHP, mySQL database implementation, HTML, CSS, and JavaScript. Functionalities include searching by Blog, Recipes, and Restaurants (neighborhoods and cuisines). Instagram.js is used to display our client’s instagram feed. Google Calendar API is used to integrate the Google Calendar in order to display our client’s important events. Logged-in admins have the ability to add a new blog post or recipe, edit an existing blog post or recipe post, and sort and move around images using the JQuery Sortable UI on the gallery page. Comments and inquiries can be sent to our client’s email via the contact page.


Favorites Collection

Created an online collection of my favorite works sorted by title, artist, keywords, year, and category using PHP, HTML, CSS, and JavaScript. New favorite works can be added and existing works can be searched.


Kappa Phi Lambda Rush

Created a webpage for Kappa Phi Lambda Sorority’s Fall 2016 Rush. The site displays sorority information and history, a short description for each sister, a JavaScript gallery of pictures, a detailed description and schedule for rush.


Online Photo Album

Created an online photo album of my artworks using PHP, mySQL, HTML and CSS. Functionalities include adding photos, adding albums, editing photos, editing albums, and deleting photos/albums. Search functionality is also implemented.
